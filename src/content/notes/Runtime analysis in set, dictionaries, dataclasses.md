# Runtime analysis in set, dictionaries, dataclasses
## [[Week 9 Lecture 3]]
- Store data in [[hash tables]]
- The hash table allows for a lower reach time as the program is able to directly search for the value as necessary.
- However we need data types to be [[Hashable]] to be a part of [[Sets]] or [[Dictionaries]].
- [[dataclass]] store data in [[Dictionaries]] so have the same benefits as dictionaries.
