# all or none principle
##
- One neuron will fire with the same magnitude each time. There is no in between. 
- How frequently it fires is a different matter though since that can vary.
	- If I play basketball often then the neuron related to basketball will fire more often but if I don’t play basketball then it won’t fire at all. ^4effc5
