## Complement: 
If we are working with an [[ambient set]], we can use a different notation. Supposing the ambient set is called V: 
$$ B^c$$
instead of $$ V \setminus B$$
However, this can confuse us what the actual set is as it is not directly stated that we are talking about V here. 
![[Drawing 2023-07-14 22.56.41.excalidraw]]

So avoid using this.
