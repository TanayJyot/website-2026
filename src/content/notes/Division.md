$$ Q =\{\frac{p}{q} : p,q \in \mathbb{Z} \land q \ge 1 \land gcd(p,q) = 1\} $$
GCD represents [[Greatest Common Divisor]]

**Notation**: For any pair of integers $$ n,m \in \mathbb{Z}, m \ne 0 \text{ and for } \frac{p}{q} \in Q \text{ we have } $$
$$ \frac{n}{p} = \frac{p}{q} \iff nq = mp $$
- This can be used to denote things like 1/2 = 2/4 = -4/-8, etc. \
- Formally we can define $Q \subseteq \mathbb{Z}\times \mathbb{Z}$ as follows: 
$$ Q = \{(p,q) \in \mathbb{Z} \times \mathbb{Z} : q \ge 1 \land gcd(p,q) = 1\}$$[^1]

[^1]: We should also be able to say Z $\times$ N right?
