# batch
##
- A batch contains a bunch of inputs, so if you increase the batch size then the batch would have more inputs.
- It becomes easier for the neuron to work with if you show it more samples at a time
- You can’t just show all the samples at the same time because that would mean that since the model knows each and every sample already, it would struggle if we give it anything out of that sample(because it is not trained to work with anything out of that sample).
