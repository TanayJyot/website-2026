# probability function
##

> Definition. A probability function P on a finite sample space Ω assigns to each event A in Ω a number P(A) in [0,1] such that 
> (i) P(Ω) = 1, and 
> (ii) P(A ∪ B) = P(A) + P(B) if A and B are disjoint. The number P(A) is called the probability that A occurs.

- For this we assume that the [[sample space]] is either [[finite set]] or [[countably infinite]]
- [[Probability [Pages 1-44] [2024-09-08].pdf#page=5&selection=85,0,99,30|Probability [Pages 1-44] [2024-09-08], page 5]]
- In a finite [[sample space]] you can list out all the elements as $\{\omega_1, ..., \omega_k\}$ 

### What you need
- The outcome of the [[sample space]]
- The probability of each outcome to occur.
