# path
##
- Basically two vertices are connected by different vertices and they can take a path consisting of $k$ steps to do so. 
- [[connectivity]]
- A single vertex is a valid path
- You cannot repeat the same vertex/ `item` in a path
- #csc111/question 
### Definition
- Let $u, u' \in V$ 
- Sequence of distinct vertices s.t. 
	- $v_0 = u$ and $u’ = v_k$ 
	- The vertices called $v_0, ,v_1, v_2$ are ordered s.t. $v_0, v_1$ will have to be [[adjacent]]. 
### Property
- Length of path = number of vertices - 1 $\equiv$ Length of path = number of edges
