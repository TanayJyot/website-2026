# character
##
- We need to have a single quotes`''` when declaring these.
- [[StringBuilder.setCharAt]] takes in a character and thus needs these single quotes.
- use $\Sigma$ to denote the alphabet = non empty finite set of characters.
- We don’t use compound symbols as characters (“aa“ cannot be a character because then what is “aaa” formed by?), we want to avoid creating anything that can be confusing to decipher to be a character.
