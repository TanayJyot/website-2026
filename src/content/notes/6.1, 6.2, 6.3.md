## 6.2 Objects and Object mutation 
![[Object Mutation]]
[[list.append#from 6.1, 6.2, 6.3| Using append prevents the id of the object from being changed. ]]
- So when you mutate a list using [[Variable Reassignment]] you just end up creating a new list with a new id but using [[list.append]] you [[Object Mutation | object mutate]] the list with the same [[id]] on it. 
	- Basically you make a new object each time with reassignment. 
- [[Mutability#From 6.1, 6.2, 6.3]]

- The value of an [[object]] does not change, the value of a variable can change.
