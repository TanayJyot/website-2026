#### Branching Programs:

Using the different outputs of conditions in such a way that they are similar to branches. 
A condition will create 2 branches a ‘true’ branch and a ‘false’ branch. 

- A false block is not compulsory 

a = int(input("Enter a number"))
```python

if a%2==0: 
    print("the number is even party party party")
else:
    print("aww")    
    

```


- Indented set of expressions represent a block of instructions
___
- \<condition> has a value(true or false)
- evaluate expressions in the block if condition is true.

- In most cases each set of statements only gets executed once. 
- Maximum time to run program only depends on length of program.
