- Joins all elements in the string and puts in the attribute mentioned in the command before every string.
```python 
List1 = ["H", "i"]
string1 = ''.join(List1)
print(string1)
# Outputs Hi
```
- If you put in something in the dialogue box then that will be repeated after each character. 

```python
List1 = ["H", "m", "p","h"]
string1 = 'm'.join(List1)
print(string1)
# Outputs Hmmmpmh
```
