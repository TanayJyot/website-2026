- You can change the value in these types using the index directly. 
- Like [[Tuples and Lists#Lists]]
```python
List1 = ["Hi", "Hello", "Goodnight"]
List1[2] = "Hey"
print(List1)
# Outputs ['Hi', 'Hello', 'Hey']
```
The list itself can be changed this way
