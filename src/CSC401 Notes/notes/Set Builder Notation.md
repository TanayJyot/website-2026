In certain circumstances it becomes difficult to write down all the values of sets. We use set builder notation in such cases. 
$$ \{1,2,3,4... \} = \{a : a \ge 0\}$$


# From [[MAT137/Preparation/Week 1]]

The symbol ‘:’ is read as such that. 

$$A = \{x  \in \mathbb{Z}: x^2 < 6\}$$
- Before the colon we describe the **location** of the element and after the colon we show additional constraints.
- In this form x is what we are using and is the random variable. 
___

- Alternatively, before the colon we can give the description of the set and after we can give more details like:
$$B = \{2x : x \in  A\}$$
- Here 2x is the random variable.
