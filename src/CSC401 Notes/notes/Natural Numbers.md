# Definition
The natural numbers can be defined using the [[set theory]] of [[Sets]]. If we consider that: 
$$ 0 \rightarrow \emptyset $$
$$ 1 \rightarrow \{\emptyset\} $$
$$  2 \rightarrow \{\emptyset \{\emptyset\}\}$$ $$ 3 \rightarrow \{\emptyset \{\emptyset\} \{\emptyset \{\emptyset\}\}\}$$
- If we substitute the value of the empty sets into 3, it can also be written like this. 
$$ 3 \rightarrow \{0, 1, 2\}$$

We can basically keep going and represent all the numbers as sets. So numbers in a way can be said then to be just sets and more sets.

We can keep going and then define the **natural numbers** like this. 

$$ n = \{0,1,..., n-1\} $$
___
So, $$\mathbb{N} = \{0,1,2,..., n, ...\}$$
is an infinite set of all natural numbers.
