# null hypothesis
##
- [[p-value]]
- this is basically when whatever you do has no effect on the outcome 
- if what you do has no effect then things are happening by random chance → uniform distribution
- The slides give the example of "The sample is drawn from a normal distribution with some fixed mean" as a null hypothesi
	- this works for language since language doesn’t follow normal distribution but [[Zipf’s law]] which is why if our model turns out to be normally distributed then it isn’t working
	- also why we need complicated things to model language like [[smoothing]] and [[Zipf-Mandelbrot Equation]]
