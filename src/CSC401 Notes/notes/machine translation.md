### Tried Methods
#### RNNs
we can’t just use RNNs to do Machine Translation because the mapping between 2 different languages isn’t 1 to 1 so some words may appear later and source length and target length aren’t the same 

#### Direct Translation
-  Translate each word from one language to another directly by a bilingual directionary
- word ordering would have to be the same for this
