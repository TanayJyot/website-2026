To sum 1-100 you can simply add them together like this. [[Johann Karl Friedrich Gauss]] came up with this in kindergarten. 
![[Pasted image 20230728200545.png]]

This is basically saying that the number of iterations of this formula is half of the total number in a sense. So you take that which is $\frac{n}{2}$ and multiply it by $(n+1)$ simply because that is what each of the iterations add up to. So we can say that the summation formula is nothing but:$$\frac{n(n+1)}{2}$$
