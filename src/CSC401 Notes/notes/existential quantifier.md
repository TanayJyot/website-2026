# Existential quantifier
> This quantifier is used to express the statement that a given predicate is true for **at least one** element in a set. It is denoted by the symbol ∃, which is read as "there exists". For example, the statement "There exists a real number x such that x2=2" means that the predicate x2=2 is true for **at least one** real number x.

- $\exists$ indicates that for this particular predicate to work, a certain kind of [[bounded variables]] must exist.
