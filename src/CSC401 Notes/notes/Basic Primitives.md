#### Basic Primitives
- You can compute anything with 6 operations:
	1. move left
	2. move right 
	3. scan 
	4. read
	5. write 
	6. do nothing
- You wanna create abstract methods to create new primitives because it will be very difficult to execute everything using only the 6 primitives.
- That’s why modern languages have way more primitives and more convenient ones. 
- Anything programmable in one language is programmable in any other language.
