# height of a tree
##
- The height would be the longest route of the tree
- The height of a subtree would be the height of the tree - 1. [[subtree]]
- Can be measured by either the number of edges or the number of nodes. 
- Number of edges = Number of nodes - 1
### Height of an empty tree
- Considered to be -1 when we are calculating [[balance factor]]s by number of edges.
