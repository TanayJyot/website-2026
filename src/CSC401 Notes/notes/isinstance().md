# isinstance()
##
It can used with [[all()]] and [[List Comprehension]] to check for more values. [^1]
- Check whether an [[object]] is a part of a particular [[class]] or any of its [[subclass]]

```python
>>> type(my_stack is Stack)
False
>>> isinstance(my_stack, Stack)
True 
```

##
- Can also be used to check if a class inherits from an [[interface]].
