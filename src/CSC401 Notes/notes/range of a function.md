# range of a function
##
The [[range of a function]] of F is :
$$Range(f) = Im(f) = f[A]=\{f(a): a \in A$$ ^d1678f


Notice that $Range(f) \subseteq Codomain(f)$ In fact, the range is the **smallest** possible codomain.  

## The range of a function
This is a set of all possible outputs of that function. This is similar to the codomain but it is difficult to figure out the range in every situation so we have the  codomain.
